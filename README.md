# Usage
```
% cd ~/temp/glitch/glitch
% python -m glitch -i `lena.jpg`
```
or  
`python -m glitch.glitch -i 'lena.jpg' -o 'glitched.jpg' -n 3`  
or  
`python -m glitch.glitch -i 'lena.jpg' maximum`  
or  
`python -m glitch.glitch -i 'lena.jpg' maximum hard`
